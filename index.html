Mis Habilidades

Como mencioné, estudié programación y desarrollo de software además de Diseño Web. Lo siguiente es lo que puedo hacer.

HTML el lenguaje que define la estructura de la Web. Puedo crear paginas con este lenguaje junto a CSS y JavaScript para que la interfaz sea lo más estética y dinámica posible

CSS el lenguaje que da vida a la páginas y aplaciones web. Si quieres tener una interfaz muy atractiva no dudes en pedirmela.

JavaScript el lenguaje de programación número uno en la Web. Tener solo HTML Y CSS no basta para tener una interfaz completa. Por eso usamos este lenguaje, además que con este podemas crear aplicaciones funcionales.

PHP otro lenguaje de programación muy importante en la Web. Si tienes un negocio, este lenguaje es muy necesario para ofrecer los productos que tienes trayendo la información necesaria desde una base de datos.

Python un lenguaje muy reconocido con el cual podemos crear aplicaciones web, analizar datos, automatizar operaciones y crear aplicaciones empresariales fiables y escalables.

Git, una herramienta de control de versiones muy necesaria con la cual podemos trabajar en equipo y dividir las tareas para ser mas productivos. Asi es tambien puedo trabajar en equipo para tu empresa.
